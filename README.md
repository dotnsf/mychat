# MyChat

## Overview

Quite simple Web Chat system using Node.js + Express + EJS( without DB ).

This can be a training application for Node.js.

## Pre-requisite

- Node.js

## Install / Setup

- `$ git clone https://github.com/dotnsf/mychat.git`

- `$ cd mycht`

- `$ npm install`

- `$ node app`

- Browse http://**.**.**.**:(port)/ with web browser.
